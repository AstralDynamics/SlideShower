    Get Started

    To get started download the SlideShower zip and extract the contents into your web directory. You should have the following files:

    • js/
      • SlideShower.js
      • lib/
        • jquery.min.js
    • css/
      • slideshower.css
    • img/
      • logo.png
      • github_icon.png

    Add the script to the page you want to use the slideshow on. <script type="text/javascript" src="js/SlideShower.js"></script>

    Then copy and paste this HTML into your page.

    <div id="slideshower" class="slideshower">
  <div class="sld-overlay">
    <img class="logo" src="img/logo.png"></img>
    <a href="https://github.com/AstralDynamics/SlideShower">
      <img class="git" src="img/github_icon.png"></img>
    </a>
    <div id="sld-progress" class="progress-bar"></div>
  </div>
</div>

    Now What?

    Now you are ready to load your array of images into the slideshow, you can do this by calling the following functions.

    This takes the array of URL and creates an array of image objects to use in the slideshow.

    SlideShower.initImages(["image.png", "another.png", "more.jpeg"]);
SlideShower.loadImages();

    Then, when the DOM is ready, you can bind events to the controls and append the images to the slideshow with the following functions.

    $(document).ready(function () {    
    SlideShower.bindControls();
    SlideShower.displayImages();
});

    And you're away! Watch a live demo here.
